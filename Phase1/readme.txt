Instructions to build the spider and test program:

General:
Both spider and test program are written and run in Python version 3.9.13
Database package: sqlite3

** please make sure you are in the "Phase1" folder before running the program below
Run the follow command to Phase1 directory: cd ./Phase1 

spider program:
1. Run the following command in terminal: python main.py
2. A database file named "search_engine.db" should be formed

test program:
1. Run the following command in terminal: python test.py
2. A output txt file named "spider result.txt" should be created
